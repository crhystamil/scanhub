Upload with Curl

It's also possible to upload data directly to this scanhub from the server that contains the XML using the 'curl' command-line tool.

curl -F nmap_xml=@YOUR_NMAP_XML_FILE "https://scanhub.shodan.io/repository/upload/kio-networks?key=2kis2ZSGEM06jzoPZi8zUSsDIYLk3Aqt"

For example, if you've stored your Nmap XML results in the current directory with a filename of 'nmap-results.xml':

curl -F nmap_xml=@nmap-results.xml "https://scanhub.shodan.io/repository/upload/kio-networks?key=2kis2ZSGEM06jzoPZi8zUSsDIYLk3Aqt"
